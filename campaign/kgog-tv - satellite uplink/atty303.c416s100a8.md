## XA

```
GRAB 300
LINK 800
LINK 799
SEEK 1
COPY F X
REPL AZIM
SEEK 1
COPY F X
REPL ELEV

NOTE FREQ
SEEK 1
COPY F #FREQ

WIPE
HALT

MARK AZIM
LINK 800
MARK AZIM_LOOP
SUBI X #AZIM T
FJMP AZIM_DONE
COPY T #MOTR
JUMP AZIM_LOOP
MARK AZIM_DONE
COPY 1 M

MARK ELEV
LINK 801
MARK ELEV_LOOP
SUBI X #ELEV T
FJMP ELEV_DONE
COPY T #MOTR
JUMP ELEV_LOOP
MARK ELEV_DONE
COPY 1 M
```

## XB (LOCAL)

```
GRAB 301
LINK 800

REPL SECURE
REPL WRITER

SEEK 1

MARK READER
TEST EOF
TJMP DONE
SUBI F 9999 X
ADDI X M X
TEST X < 0
FJMP CLAMP
ADDI X 9999 X
JUMP WRITE
MARK CLAMP
SUBI X 1 X
NOOP
MARK WRITE
COPY X M
JUMP READER

MARK DONE
COPY -1 M
WIPE
LINK -1
HALT


MARK WRITER
MAKE
@REP 4
NOOP
@END
MARK WRITE_LOOP
@REP 5
NOOP
@END
COPY M X
TEST X < 0
TJMP WRITE_DONE
COPY X F
JUMP WRITE_LOOP
MARK WRITE_DONE
KILL
LINK 799
SEEK -9999
MODE
VOID M
VOID M

MARK WW
TEST EOF
TJMP WIPE
COPY F #DATA
JUMP WW
MARK WIPE
WIPE

MARK SECURE
GRAB 199
MARK SECURE_LOOP
COPY F M
TEST EOF
FJMP SYNC

SEEK -9999
JUMP SYNC2

MARK SYNC
NOOP
NOOP
MARK SYNC2

JUMP SECURE_LOOP
```
